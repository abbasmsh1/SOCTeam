#### **If DDoS Confirmed:**

   - Consider **anycast routing** to distribute attack traffic.

🔹 **Document Attack Details:**
   - Attack type (SYN flood, UDP flood, etc.).
   - Peak bandwidth (Gbps) and duration.
   - Affected services (e.g., web app, API, DNS).

#### **If Infiltration Suspected:**
🔹 **Isolate Affected Systems:**

   - Revoke compromised credentials (rotate keys, disable accounts).

🔹 **Hunt for Persistence:**
   - Check for **scheduled tasks, WMI subscriptions, or startup scripts**.
   - Review **registry modifications** (e.g., `Run` keys).

🔹 **Collect Forensic Evidence:**
   - Memory dumps (`volatility`, `Rekall`).
   - Disk images (`FTK`, `Autopsy`).
   - Network captures (`PCAP`, `Zeek logs`).

#### **4. Hunt for Related Threats**
   - **Check for:**
     - **Malware** (e.g., Mimikatz, RDP worms like **BlueKeep**).
     - **Persistence mechanisms** (scheduled tasks, new services, registry modifications).
     - **C2 Beacons** (unusual outbound connections to known malicious IPs).

#### **5. Containment & Eradication (If Confirmed Breach)**
   - **Isolate affected systems** (network segmentation, disable RDP/SSH if needed).
   - **Reset all credentials** associated with the targeted account(s).
   - **Patch vulnerable services** (e.g., unpatched RDP, weak SSH configs).
   - **Deploy EDR/XDR rules** to detect similar attacks in the future.

#### **6. Long-Term Mitigations**
   - **Enforce MFA** on all remote access (RDP, VPN, OWA).
   - **Implement account lockout policies** (e.g., 5 failed attempts = lockout).
   - **Enable Windows Event ID 4625** (failed logins) and **Linux auth.log** monitoring.
   - **Deploy a WAF** if web apps are targeted.
   - **Threat Hunting:** Proactively search for brute force patterns in historical logs.

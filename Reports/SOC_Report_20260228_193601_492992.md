### **4. Longer-Term Mitigations**
| **Action** | **Details** |
|------------|------------|
| **Enforce MFA** | Critical for RDP, VPN, and admin accounts. |
| **Rate-limiting** | Block IPs after 5–10 failed attempts. |
| **Password Policy** | Enforce complexity + regular rotation. |
| **Disable Legacy Auth** | Block basic auth (e.g., for O365/Exchange). |
| **Honeypot Accounts** | Detect brute force early (e.g., fake "admin" user). |
| **SIEM Alert Tuning** | Reduce false positives (e.g., ignore internal IPs). |

```
#### **Block IP on Windows (Firewall):**
```powershell
New-NetFirewallRule -DisplayName "Block Brute Force IP" -Direction Inbound -RemoteAddress [ATTACKER_IP] -Action Block

#### **Check Failed Logins (Linux):**
